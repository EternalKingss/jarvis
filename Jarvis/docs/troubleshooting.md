# Troubleshooting Jarvis

This guide lists common problems and steps to resolve them.

## Startup Issues
- **Configuration file not found**: Run `python main.py --configure` to create one.
- **Missing API keys**: Ensure OpenAI and ElevenLabs keys are stored in `jarvis_config.ini`.
- **No microphone detected**: Verify your audio hardware and permissions.


## Network Errors
- If online services fail, check your internet connection and proxy settings.
